# library-
# 📚 Library Management System (OOP)

## 🧠 Project Overview

This project implements a simple Library Management System using Object-Oriented Programming (OOP) concepts in Python.

---

## 🎯 Objective

To demonstrate core OOP principles such as:

* Classes and Objects
* Encapsulation
* Abstraction
* Code reusability

---

## ⚙️ Features

* Add books to the library
* Register users
* Issue books to users
* Return books
* Display available and issued books

---

## 🏗️ System Design

### 📘 Book Class

Represents a book with:

* ID
* Title
* Author
* Issue status

### 👤 User Class

Represents a user with:

* User ID
* Name
* Borrowed books list

### 🏢 Library Class

Handles:

* Book management
* User management
* Issue & return operations

---

## 🔍 Key Concepts Used

* Class & Object creation
* Constructor (`__init__`)
* Methods & attributes
* Lists and dictionaries
* Conditional logic

---

## 💡 Key Learnings

* How to structure real-world problems using OOP
* Managing relationships between objects
* Writing modular and reusable code

---

## 🛠 Tools Used

* Python

---

1. Install Python
2. Run the file:

```
python library_management.py
```

This project demonstrates how OOP can be used to model real-world systems like a library and manage operations efficiently.
